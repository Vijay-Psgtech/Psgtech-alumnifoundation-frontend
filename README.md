# PSG Tech Alumni Association

A modern, responsive web platform for the PSG Tech Alumni Association built with React, Vite, and Tailwind CSS.


## 🎯 Features

### Core Features
- ✅ **Banner Section** - Animated hero with background image and overlay effects
- ✅ **About Page** - Association overview with statistics and mission highlights
- ✅ **Vision & Values** - Strategic components with animations
- ✅ **Objectives** - Interactive expandable cards with 8 core objectives
- ✅ **Contact Us** - Fully responsive contact form with location map
- ✅ **Navigation** - Sticky navbar with dropdown menus and Donate CTA

### 💳 DonationGate (Payment System)
- ✅ **Form Data Persistence** - localStorage integration for seamless UX
- ✅ **KYC Verification** - Multi-step donor verification process
- ✅ **Razorpay Integration** - Real payment gateway for INR donations
- ✅ **Dynamic Amounts** - Support for ₹100 to ₹10,00,000 donations
- ✅ **Form Validation** - PAN, Mobile, Email, and document validation
- ✅ **Transaction History** - Saves donation data with Razorpay IDs
- ✅ **Email Receipts** - Automatic receipt generation (ready for backend)

### 🎨 Design System
- Dark theme with glassmorphism effects
- Blue-to-Purple gradient palette
- Staggered animations with Framer Motion
- Fully responsive (mobile, tablet, desktop)
- Touch-friendly interface

## 🛠️ Tech Stack

- **Frontend:** React 19.2.0, Vite 7.2.2, Tailwind CSS 4.1.17
- **Animations:** Framer Motion 12.23.24
- **Icons:** Lucide React 0.554.0
- **Routing:** React Router DOM 7.9.6
- **Payment:** Razorpay Checkout
- **Deployment:** Docker + Nginx (SPA routing)
